Let's make some doggie games!



 Start making games!

What is it?

This is an interactive playground for writing treat-dispensing games that work in the Doggie and Me app (using a PetTutor treat dispenser) - and also for learning how to write them in the first place!

How does it work?

There are lots of examples to learn from. Each example has an explanation at the top, and a reference to all the commands at the bottom. You can run and change the example as much as you like, then deploy your version to the Doggie And Me app.

(If you just want to see what's possible, there's also a link to a big reference page on the top-right.)

What do I do?

Not sure how to start? The button below will take you to the first very simple example - play around with it, then try the next example, then the next, and so on!

Show me the games!

This playground is based on the Pencil Code Gym, and the Doggie And Me games also use Pencil Code's turtle graphics. So if you want to learn even more programming, check out Pencil Code!
