Reference

Welcome to the Code Gym reference library.

To use this library:

  1. Click on any topic on the left. Feel free to browse any topics you would like to learn more about.
  2. Try out the examples on the pages. Nothing is saved, so you can try anything you like.
    • If the example has a   blue   background, it only works in Doggie And Me projects.
    • If it has a white background with a grid, it will work in any Pencil Code project.
  3. Copy the ideas that you like back into your Code Gym projects. Project areas are linked in the menu at the top.

Happy browsing.
